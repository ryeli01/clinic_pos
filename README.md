## Getting Started

To work on this project locally:

1. Clone the repository to your local machine using your terminal:
   `git clone https://github.com/eophyrhim/clinic_pos.git`
2. Open the `clinic_pos` folder in your preferred code editor.
3. Open `index.html` in your web browser to view the current state of the application.
